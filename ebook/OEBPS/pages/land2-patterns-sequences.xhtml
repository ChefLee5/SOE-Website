# Word Pronunciation In the Story… ASL Sign 🤟 My Language
1 Pattern /PAT-ern/ A pattern repeats in a predictable way. Fingerspell or sign each word
2 Repeat /rih-PEET/ The pattern of colors repeats: red, blue, red, blue. Fingerspell REPEAT
3 Sequence /SEE-kwens/ A sequence has numbers or objects in a specific order. Fingerspell or sign each word
4 Next /NEKST/ What comes next in the pattern: 2, 4, 6, ___? Fingerspell NEXT
5 Predict /prih-DIKT/ Predict the next shape in the sequence. Fingerspell or sign each word
6 Sort /SORT/ Sort the buttons by size from smallest to largest. Fingerspell SORT
7 Classify /KLAS-ih-fy/ Classify the shapes into groups. Fingerspell or sign each word
8 Arrange /uh-RAYNJ/ Arrange the numbers in order from least to greatest. Fingerspell or sign each word
9 Organize /OR-guh-nyz/ Organize your data before making a chart. Fingerspell or sign each word
10 Category /KAT-uh-gor-ee/ Put each animal into the correct category. Fingerspell or sign each word
11 Rule /ROOL/ The rule for this pattern is "add three." Fingerspell RULE
12 Skip count /SKIP KOWNT/ Skip count by fives: 5, 10, 15, 20. Fingerspell or sign each word
13 Growing pattern /GROH-ing PAT-ern/ A growing pattern gets bigger each time. Fingerspell or sign each word
14 Shrinking pattern /SHRINK-ing PAT-ern/ A shrinking pattern gets smaller each time. Fingerspell or sign each word
15 Symmetry /SIM-eh-tree/ This design has symmetry — both sides match. Fingerspell or sign each word
16 Repeating /rih-PEE-ting/ The repeating pattern goes: star, moon, star, moon. Fingerspell or sign each word
17 Core /KOR/ The core of the pattern is the part that repeats. Fingerspell CORE
18 Extend /ek-STEND/ Extend the pattern by adding more shapes. Fingerspell EXTEND
19 Function /FUNK-shun/ A function is a rule that changes one number to another. Fingerspell or sign each word
20 Input /IN-poot/ The input goes into the machine. Fingerspell INPUT
21 Output /OWT-poot/ The output comes out after the rule is applied. Fingerspell OUTPUT
22 Table /TAY-bul/ Use a table to show the input and output. Both forearms stack, form flat surface
23 Variable /VAIR-ee-uh-bul/ A variable is a letter that stands for an unknown number. Fingerspell or sign each word
24 Missing number /MIS-ing NUM-ber/ Find the missing number: 3 + ___ = 10. Fingerspell or sign each word
25 Algebra /AL-juh-bruh/ Algebra uses letters and numbers to solve equations. Fingerspell or sign each word

Octavia's Tip:
